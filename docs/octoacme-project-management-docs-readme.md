## Purpose
This README provides a single entry point for OctoAcme project management processes, with a short summary and direct links to all process documents in `docs/`.

## OctoAcme Project Management Process Summary
OctoAcme uses a structured, iterative project management approach focused on clarity, collaboration, and continuous improvement.

- Start with a clear project initiation (problem statement, stakeholders, objectives, and initial scope).
- Plan by defining backlog, milestones, dependencies, roles, execution approach, and communication cadence.
- Track execution with regular check-ins, progress reviews, risk/issue updates, and built-in quality checks.
- Release using gate criteria, verification, deployment readiness, and stakeholder notifications.
- Retrospect after delivery to capture lessons, action items, and process improvements.

Core principles: customer focus, iterative delivery, clear ownership, data-informed decisions, psychological safety.
